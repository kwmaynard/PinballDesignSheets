# PinballDesignSheets
Printable sheets for designing pinball tables. The tables include a typical, Italian bottom third. There is a grid for each table in cyan. Each square represents 1 square inch. Pinball table size is 20.25" by 45".
There are 4 sizes of the print out.
1. Full Size: The print is 24" by 48" and represents a full sized playfield.
2. Legal: The print is on legal sized paper (8.5" by 14"), and the playfield is at 1/3.46 scale.
3. Letter: The print is on letter sized paper (8.5" by 11"), and the playfield is at 1/4.5 scale.
4. Journal: The print is for a 6" by 9" journal, and the the playfield is at 1/5.625 scale.
